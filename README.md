# Лабораторная работа №6

Тема: "Стандартная библиотека C++. Последовательные и ассоциативные контейнеры. Обобщенные алгоритмы".

В проекте реализован вариант 4: шаблонный класс с контейнером `std::vector`, копированием массива, случайным перемешиванием, вычислением среднего, поэлементным умножением и возведением результата в квадрат с использованием стандартных алгоритмов, предиката и объекта-функции.

## Сборка

```powershell
cmake -S . -B build -G "Visual Studio 17 2022"
cmake --build build --config Release
```

## Запуск

```powershell
.\build\Release\lab6_stl_algorithms.exe
```
